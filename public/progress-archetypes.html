Progress — Archetypes × Layouts

Eight user archetypes run through the personalization engine, each rendered in layouts A (Hero+Tabs), B (Dashboard Cards) and C (Story Feed). Same rules, same cards, three presentations.

Reading guide. Each archetype shows engine output — which cards are included, their order, and their framing variant — then renders it in all three layouts. Cards that are hidden for this user don't appear at all; they'd still be reachable via "Show more". The single-line subheader under the title is the only explicit tell that personalization is happening.

What to notice across the 8. The same three layouts can carry radically different content: a post-partum user never sees a PR chart; a beginner never sees a trend line; an injured user never sees the missing lifts as a drop; a perimenopausal user never sees a week-over-week dip as failure. The cards change, the framing changes, the time window changes — and the user is never told any of it was hidden. Taps on "What I want to see" open the override panel.
